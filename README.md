# Assorted_Tools
A collection/monorepo of tools and smaller projects to demonstrate (and remember) how to use a variety technologies mostly i/o, apis, and other services. The tools I'm mainly using here are Python, Typescript, and Go. The ideas here should be sound, but use at your own risk.

> This project is not intended to be built or distributed.

## Layout

This collection of tools is essentially a few different projects that host a variety of different examples. The assets folder contains examples files and small systems to simplify creating scenarios.
